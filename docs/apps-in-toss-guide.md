# 앱인토스(Apps in Toss) 개발자 가이드 조사 보고서

> 조사일: 2026-02-24
> 출처: https://developers-apps-in-toss.toss.im/

---

## 1. 플랫폼 개요

앱인토스는 파트너사가 개발한 서비스를 토스 앱 내부에서 "앱인앱(App-in-App)" 형태로 노출할 수 있는 플랫폼.

- 누적 토스 사용자: 3,000만 명
- 지원 기술: WebView SDK, React Native SDK, Unity(WebGL 포팅), Cocos
- 빌드 결과물을 콘솔에 업로드하면 토스 CDN 서버에서 자동 호스팅
- Android 최소 버전: Android 7
- iOS 최소 버전: iOS 16
- 이용 대상: 만 19세 이상만

---

## 2. 서비스 오픈 프로세스 (5단계)

| 단계 | 내용 | 소요 기간 |
|------|------|-----------|
| 1. 등록 | 앱인토스 콘솔에서 앱 등록 | 즉시 |
| 2. 디자인 | App Builder(Figma) 또는 TDS 활용 | - |
| 3. 개발 | SDK/API 연동, 샌드박스 테스트 | - |
| 4. 검수 | 운영/기능/디자인/보안 4단계 심사 | 영업일 기준 최대 3일 |
| 5. 출시 | 콘솔에서 출시 버튼 클릭 시 즉시 배포 | 즉시 |

게임 출시 총 소요 기간: 약 2~4주 (비게임은 2~3개월)

## 3. 콘솔 등록 절차 및 준비물

### 3-1. 사전 요건
- 만 19세 이상
- 본인 명의 토스앱 필요
- 사업자 등록 없이도 개발/배포 가능 (단, 수익화·토스 로그인 사용 불가)

### 3-2. 앱 기본 등록 정보

| 항목 | 요건 |
|------|------|
| 앱 로고 | 600×600px PNG, 배경색 필수 (투명 불가), 각진 정사각형 |
| 앱 이름 | 서비스 특성이 드러나는 한글명 |
| appName | intoss:// 딥링크에 사용, 한번 설정 후 수정 불가 |
| 사용 연령 | 현재 19세 이상만 지원 |
| 고객센터 | 이메일, 연락처, 채팅 상담 주소 필수 |

### 3-3. 스토어 노출용 에셋

| 항목 | 규격 |
|------|------|
| 정방형 썸네일 | 1000×1000px |
| 가로형 썸네일 | 1932×828px |
| 스크린샷 | 세로 3장 이상 또는 가로 1장 이상 |
| 부제 | 간결한 설명 (느낌표 사용 불가) |
| 상세 설명 | 사용자 경험 중심으로 구체적으로 기술 |
| 검색 키워드 | 앱 관련 예상 검색어 |

### 3-4. 게임 전용 필수 항목 (중요)

**게임 등급분류 - 필수**

아래 중 하나 이상 보유해야 함:
- 오픈마켓 등급: 앱스토어, 구글 플레이, 원스토어, MS스토어 URL
- 게임물관리위원회(GRAC) 심의 증명서 PDF
- 제출 정보: 등록자명, 자체등급분류사업자명, 분류일자, 분류번호, 이용등급, 내용정보, 인감/사인 이미지

우리 게임이 이미 MS스토어에 배포된 경우 MS스토어 URL 제출로 충족 가능.

## 4. 개발 요건 (WebView 방식)

### 4-1. 기술 스택 요건

우리 게임(React+Vite)은 WebView 방식으로 등록 가능.

**필수 설치 패키지:**
```bash
npm install @apps-in-toss/web-framework
npx ait init
```

**설정 파일 (granite.config.ts):**
```typescript
appName: 'game-app-name'    // 콘솔 앱 이름과 동일하게
brand.displayName            // 화면 노출 앱명
brand.icon                   // 앱 로고 URL
brand.primaryColor           // 대표 색상 #RRGGBB
web.host                     // 개발 서버 호스트
web.port                     // 포트 번호 (기본 5173)
```

**빌드:**
```bash
granite build   # .ait 번들 파일 생성 (100MB 이하 필수)
```

### 4-2. TDS(토스 디자인 시스템) 적용 규칙

- **비게임 WebView 미니앱**: TDS 필수 적용
- **게임 미니앱**: TDS 적용 예외 (게임 고유 UI 허용)
- TDS는 라이트 모드만 지원
- 설치 패키지: `@toss/tds-mobile` (버전 1.0.0 이상)

### 4-3. 번들 크기 제한
- 압축 해제 기준 100MB 이하
- 대용량 에셋은 CDN/외부 스토리지로 분리, 레이지 로딩 필수

### 4-4. 테스트 환경
- 앱인토스 샌드박스 앱에서 테스트 필수 (로컬 브라우저 테스트 불가)
- iOS: `intoss://{appName}` 스킴으로 접근
- Android: adb 포트 포워딩 후 접근

### 4-5. 기타 기술 제약
- iframe 사용 불가 (YouTube 삽입 예외)
- CORS: 서버에 앱인토스 도메인 허용 필요
- mTLS: 서버간 API 통신 시 필수
- OS 백 제스처 비활성화 필수 (게임)

## 5. 게임 출시 검수 체크리스트 (상세)

검수는 운영/기능/디자인/보안 4단계로 진행.

### 5-1. 접근 및 로딩
- [ ] 미니앱이 10초 이내에 최초 화면까지 정상 열림
- [ ] 앱 번들 100MB 이하

### 5-2. 사운드
- [ ] 배경음악/효과음/햅틱 정상 동작
- [ ] 사용자가 오디오 온/오프 제어 가능
- [ ] 백그라운드 이동 시 오디오 즉시 중지
- [ ] 포그라운드 복귀 시 오디오 정상 재개
- [ ] 인앱결제 진입 시 음악 일시 정지
- [ ] 광고 재생 중 배경음악 정지 및 광고 종료 후 재개

### 5-3. UX 및 내비게이션
- [ ] 우상단 닫기 버튼 정상 동작 및 항상 표시
- [ ] Safe Area 준수 (iOS Dynamic Island, 노치 침범 금지)
- [ ] 하단 시트가 진입 시 자동으로 열리지 않음
- [ ] 모든 화면에서 종료 방법 제공
- [ ] OS 백 제스처 비활성화
- [ ] CTA 버튼이 결과 행동을 명확히 표시
- [ ] 앱 종료 시 확인 모달 제공

### 5-4. 게임 로그인 및 데이터
- [ ] 사용자 ID 값 정확하게 식별 및 저장
- [ ] 플레이 기록(랭킹, 레벨, 스테이지) 유지
- [ ] 세션/앱 재시작 후에도 데이터 보존

### 5-5. 서비스 운영
- [ ] 인게임 화면 풀스크린 구현
- [ ] 가로/세로 방향 설정 정상 동작
- [ ] 금지 콘텐츠(불법, 성인 소재) 없음
- [ ] 모든 UI 컴포넌트 정상 동작
- [ ] 2초 이내 인터랙티브 응답
- [ ] 권한 요청 전 사용자 동의 획득
- [ ] 권한 거부 시에도 앱 기본 기능 유지
- [ ] 비정상적인 네트워크/메모리 급증 없음
- [ ] 핵심 게임플레이(점수, 스테이지 클리어) 오류 없음

### 5-6. 인앱 광고 (적용 시)
- [ ] 예측 불가능한 순간에 광고 표시 금지
- [ ] 광고 완전 표시 후 앱 복귀
- [ ] 사전 로드된 광고 사용 (실시간 로딩 금지)
- [ ] 리워드 광고 시청 완료 시 보상 지급

### 5-7. 리더보드 QA (앱인토스 게임 프로필 연동 시)
- [ ] 게임 종료 후에만 점수 제출 API 호출
- [ ] 동일 플레이에서 중복 제출 방지
- [ ] 네트워크 오류 시 재시도/대체 흐름
- [ ] 리더보드 정상 표시 (오버레이 겹침 없음)
- [ ] 포그라운드 복귀/화면 회전 시 UI 안정성

### 5-8. 모든 게임 공통 필수
- [ ] **게임 등급 정보 화면 제공** (콘솔에 등급분류 정보 입력 시 자동 제공)

## 6. 수익화 옵션

### 6-1. 인앱 광고 (IAA)

| 광고 유형 | 설명 |
|-----------|------|
| 전면 광고 | 화면 전체에 노출, 별도 리워드 없음 |
| 리워드 광고 | 시청 후 보상 제공 |
| 배너 광고 | 상단 또는 하단 고정 노출 |

수수료: 15% (현재 베타 기간 중 0%)
정산: 매월 말일에 직전 월 수익 입금
성과 데이터: 매주 월요일 16시 이후 업데이트

### 6-2. 인앱 결제 (IAP)

| 항목 | 내용 |
|------|------|
| 소모성 아이템 | 코인, 아이템 등 사용 후 소진 |
| 비소모성 아이템 | 광고 제거, 프리미엄 기능 등 영구 |
| 최소 가격 | 400원 (공급가 기준) |
| 최대 가격 | 1,400,000원 |
| 가격 단위 | 10원 단위 |
| 수수료 구조 | 앱마켓 수수료 15% + 토스 수수료 5% |
| 환불 정책 | Apple/Google 기준 따름 |

상품 이미지: 1024×1024px 필수

### 6-3. 토스페이

실물 상품/서비스 결제에 적용. 게임 아이템에는 적용 불가 (IAP 사용).

### 6-4. 금지 수익화
- 현금성/환금성 이벤트 불가
- 게임 아이템, 기프티콘, 상품권 현금화 불가
- 토스포인트 전환 이벤트 불가

## 7. 디자인 가이드

### 7-1. TDS 게임 예외 원칙
게임 미니앱은 TDS 적용 의무에서 예외이나, 아래 항목은 필수 준수:
- 앱인토스 전용 네비게이션 바 컴포넌트 사용
- 탭바 사용 시 토스 제공 플로팅 형태 사용 필수

### 7-2. 브랜딩 에셋 규격

| 에셋 | 규격 |
|------|------|
| 앱 로고 | 600×600px, PNG, 배경 필수, 각진 정사각형 |
| 브랜드 컬러 | 6자리 헥스코드 (#RRGGBB) |
| 정방형 썸네일 | 1000×1000px |
| 가로형 썸네일 | 1932×828px |
| IAP 상품 이미지 | 1024×1024px |

### 7-3. 모드 지원
- TDS는 라이트 모드만 지원
- 다크 모드 개발해도 출시 가능 (게임은 예외)

### 7-4. Safe Area
- iOS Dynamic Island, 노치, 카메라 홀 침범 금지
- 상단/하단 빈 여백 없이 풀스크린 구현

## 8. 우리 게임(React+Vite) 출시 준비 체크리스트

### Phase 1: 등록 준비 (사전)
- [ ] 앱인토스 콘솔 가입 및 워크스페이스 생성
- [ ] 앱 로고 제작 (600×600px PNG, 배경색 포함)
- [ ] 정방형 썸네일 제작 (1000×1000px)
- [ ] 가로형 썸네일 제작 (1932×828px)
- [ ] 스크린샷 준비 (세로 3장 이상)
- [ ] 앱 이름/부제/상세 설명 작성
- [ ] 검색 키워드 목록 작성
- [ ] 고객센터 이메일/연락처 준비
- [ ] MS스토어 URL 확보 (게임 등급분류 대체)

### Phase 2: 개발 연동
- [ ] `@apps-in-toss/web-framework` 패키지 설치
- [ ] `npx ait init` 실행 및 granite.config.ts 설정
- [ ] 번들 크기 100MB 이하 확인 (대용량 에셋 CDN 분리)
- [ ] 우상단 닫기 버튼 구현 (항상 표시)
- [ ] OS 백 제스처 비활성화 처리
- [ ] 앱 종료 시 확인 모달 구현
- [ ] Safe Area 적용 (Dynamic Island 침범 방지)
- [ ] 10초 이내 최초 화면 로딩 달성
- [ ] 백그라운드 이동 시 오디오 제어 구현
- [ ] 풀스크린 게임 화면 구현

### Phase 3: 테스트
- [ ] 앱인토스 샌드박스 앱에서 실기기 테스트 (필수)
- [ ] iOS 기기 테스트 (Dynamic Island 포함)
- [ ] Android 기기 테스트 (adb 포트 포워딩)
- [ ] 네트워크 온/오프 상태 테스트
- [ ] 포그라운드/백그라운드 전환 테스트
- [ ] 핵심 게임플레이 동작 검증
- [ ] 메모리/네트워크 사용량 프로파일링

### Phase 4: 검수 제출
- [ ] 게임 등급분류 정보 콘솔 입력 (MS스토어 URL 또는 GRAC 증명서)
- [ ] 리더보드 설정 (사용 시)
- [ ] 검수 요청 제출
- [ ] 영업일 기준 최대 3일 대기

### Phase 5: 수익화 설정 (출시 후)
- [ ] 사업자 정보 등록 (수익화 필수 조건)
- [ ] 정산 정보 입력 (통장 사본 준비)
- [ ] 인앱 광고 연동 (전면/리워드/배너 선택)
- [ ] 인앱 결제 상품 등록 (소모성/비소모성)

## 9. 핵심 주의사항

1. **앱인토스는 만 19세 이상만 이용 가능** - 미성년자 타겟 게임 불가
2. **사업자 없이 배포 가능** - 단, 수익화(광고/결제)는 사업자 필수
3. **iframe 완전 금지** - YouTube 삽입만 예외
4. **게임은 TDS 예외** - 단, 네비게이션 바 컴포넌트는 앱인토스 제공 것 사용
5. **게임 등급 필수** - MS스토어 기출시 URL로 대체 가능
6. **IAP 수수료**: 앱마켓 15% + 토스 5% = 총 20% 차감
7. **IAA 수수료**: 15% (베타 기간 현재 0%)
8. **초기 제휴사 혜택**: 일부 수수료 면제 프로모션 있음
