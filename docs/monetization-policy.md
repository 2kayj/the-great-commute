# 수익화 및 아이템 정책

## 커피 아이템 (생존 쉴드)

### 기능
- 캐릭터가 위험 상태(넘어지기 직전)일 때 **자동 발동**
- 쉴드 발동 시 20m 구간 동안 넘어지지 않음
- **런당 사용 제한 없음** — 커피가 있는 한 위험할 때마다 자동 발동

### 지급
- **첫 시작 시 3잔 무료 지급** (persist 저장, 재설치 전까지 유지)
- 이후 추가 지급 없음

### 유료 구매
- **3,000원에 커피 5잔** 구매 가능
- 인앱 결제 (Microsoft Store / 추후 모바일 스토어)

---

## 리워드 광고 (부활)

### 기능
- 게임 오버 시 **리워드 광고 시청 → 1회 부활** 가능
- 부활 시 현재 Day에서 이어서 플레이
- **런당 1회만** 부활 가능 (이미 구현: `usedContinue` 플래그)

### 수익 모델
- 광고 네트워크: 토스 애즈 SDK (예정)
- 리워드 광고 1회 시청 = 부활 1회

---

## 수익 구조 요약

| 항목 | 가격 | 내용 |
|------|------|------|
| 커피 5잔 | 3,000원 | 인앱 결제 |
| 부활 1회 | 무료 (광고 시청) | 리워드 광고 |

| 항목 | 상태 |
|------|------|
